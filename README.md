# 📊 Pixel BI — Conversational AI Business Intelligence Dashboard

> Ask natural language questions and get instant interactive charts — no SQL, no coding needed!

🇮🇳 India Market | 📅 2016–2026 | 📱 25 Models

---

## 🎯 Problem Statement

Non-technical business executives cannot access data insights without SQL knowledge. Data teams are overwhelmed with basic reporting requests. Business users wait days for simple dashboards.

## ✅ Our Solution

An intelligent Business Intelligence Dashboard that allows non-technical users to generate fully functional, interactive data dashboards using only natural language prompts.

---

## 🚀 Features

- 🤖 Ask questions in plain English — get instant answers
- 📊 4 Interactive Charts — Bar, Doughnut, Line, Price Comparison
- 📱 25 Google Pixel India Models — 2016 to 2026
- 📂 CSV Upload — Upload your own data and query instantly
- 🧠 Google Gemini AI — Intelligent business insights
- 🔍 Filter by Year and Model
- 📱 Phone Modal — Click any model to see full specs

---

## 🛠️ Tech Stack

| Layer | Technology |
|-------|-----------|
| Frontend | HTML5, CSS3, Vanilla JavaScript |
| Charts | Chart.js |
| AI | Google Gemini API |
| Backend | Python Flask |
| Data | JavaScript Dataset + CSV Upload |
